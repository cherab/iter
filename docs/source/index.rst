=========================
CHERAB-ITER documentation
=========================
Welcome to the CHERAB-ITER's documentation.
This package offers ITER tokamak specific features - Plasma emitter,
Plasma facing components, diagnostic configuration, etc.

Please note that this package is under development,
so this documentation is gradually updated, and welcome your contribution.
